# рџ“Њ Overview

TataBot is an AI-powered Knowledge Assistant developed during my **AI Internship at Tata Steel** under the **Learning & Development (L&D) Innovation** initiative.

The application uses **Retrieval-Augmented Generation (RAG)** and **Llama 3.3 via Groq API** to help employees quickly access organizational knowledge, including SOPs, safety procedures, troubleshooting guides, and training materials through natural language conversations.

---

# рџЋЇ Problem Statement

Employees often spend significant time searching through multiple documents for relevant information.

Common challenges include:

- Time-consuming document searches
- Knowledge scattered across multiple files
- Slow onboarding of new employees
- Difficulty finding role-specific information
- Repetitive support queries

---

# рџ’Ў Solution

TataBot combines **Large Language Models (LLMs)** with **Retrieval-Augmented Generation (RAG)** to provide fast, accurate, and context-aware answers.

Instead of relying only on the AI model, TataBot first retrieves relevant information from the knowledge base and then generates responses based on that context.

# рџ”„ Workflow

1. User asks a question.
2. Flask receives the request.
3. Relevant documents are retrieved from the knowledge base.
4. Context is sent to Groq API.
5. Llama 3.3 generates a contextual response.
6. The chatbot returns the final answer.
